# Contributing

Thanks for your interest in EnterpriseAiDataAgent.

## Before You Start

- Do not commit `.env` files or API keys.
- Run backend and frontend checks before submitting changes.
- Keep README claims aligned with implemented features.

## Suggested Checks

```bash
# Backend
pytest
ruff check backend

# Frontend
cd frontend-react
npm run test
npm run build
npm run lint
```

## Pull Requests

Please keep PRs focused and describe:

- What changed
- Why it changed
- How it was tested
