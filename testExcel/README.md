# Test Datasets — Enterprise AI Data Agent

用于 v0.3 数据接入层及后续 AI 分析功能的测试数据集。

## 数据集概览

| 文件 | 行数 | 列数 | 缺失率 | 用途 |
|------|------|------|--------|------|
| sales_data.xlsx | 1200 | 8 | ~1.4% | 销售分析、趋势预测 |
| customer_data.xlsx | 1200 | 12 | ~1.1% | 客户画像、分群分析 |
| inventory_data.xlsx | 1200 | 11 | ~1.5% | 库存管理、预警分析 |
| finance_data.xlsx | 1200 | 11 | ~1.3% | 财务分析、预算对比 |

---

## 1. sales_data.xlsx — 销售数据

企业销售流水记录，覆盖 2024-2025 年。

| 字段 | 类型 | 说明 |
|------|------|------|
| 日期 | DATE | 交易日期 |
| 地区 | VARCHAR | 华东/华南/华北等 7 大区 |
| 产品 | VARCHAR | 12 种智能硬件产品 |
| 销量 | INTEGER | 单次交易数量 |
| 销售额 | DOUBLE | 交易金额（含异常值） |
| 利润 | DOUBLE | 交易利润（含缺失值） |
| 客户类型 | VARCHAR | 企业/个人/政府/教育/中小企业 |
| 销售员 | VARCHAR | 12 名销售人员 |

**适合测试**: SQL 聚合分析、地区/产品维度切片、销售趋势图表、异常值检测

---

## 2. customer_data.xlsx — 客户数据

企业客户信息库。

| 字段 | 类型 | 说明 |
|------|------|------|
| 客户ID | VARCHAR | 唯一标识 (C000001 格式) |
| 客户名称 | VARCHAR | 客户名称 |
| 行业 | VARCHAR | 科技/金融/制造等 10 个行业 |
| 省份 | VARCHAR | 20 个省级行政区 |
| 城市 | VARCHAR | 对应城市 |
| 注册日期 | DATE | 客户注册时间 |
| 累计消费 | DOUBLE | 历史总消费（含异常值） |
| 订单总数 | INTEGER | 历史订单数 |
| 满意度评分 | DOUBLE | 1-5 分（含缺失值） |
| 是否活跃 | BOOLEAN | 当前活跃状态 |
| 信用等级 | VARCHAR | AAA/AA/A/B/C |
| 联系邮箱 | VARCHAR | 邮箱地址（含缺失值） |

**适合测试**: 客户分群、RFM 分析、行业分布、省份热力图、信用评级分析

---

## 3. inventory_data.xlsx — 库存数据

多仓库库存管理数据。

| 字段 | 类型 | 说明 |
|------|------|------|
| 商品ID | VARCHAR | 唯一标识 (P00001 格式) |
| 商品名称 | VARCHAR | 产品名称 |
| 仓库 | VARCHAR | 5 个区域仓库 |
| 库存数量 | INTEGER | 当前库存（含异常值） |
| 最低库存 | INTEGER | 安全库存阈值 |
| 最高库存 | INTEGER | 最大库存容量 |
| 单位成本 | DOUBLE | 商品成本（含缺失值） |
| 最后入库日期 | DATE | 最近一次入库时间 |
| 最后出库日期 | DATE | 最近一次出库时间（含缺失值） |
| 保质期(天) | INTEGER | 商品保质期（0=无限制） |
| 库存状态 | VARCHAR | 充足/正常/低库存 |

**适合测试**: 库存预警、周转率分析、仓库对比、成本分析、保质期管理

---

## 4. finance_data.xlsx — 财务数据

企业费用支出记录，覆盖 2023-2025 年。

| 字段 | 类型 | 说明 |
|------|------|------|
| 日期 | DATE | 支出日期 |
| 部门 | VARCHAR | 8 个企业部门 |
| 费用类别 | VARCHAR | 8 类费用（人力/营销/研发等） |
| 金额 | DOUBLE | 实际支出（含异常值） |
| 预算 | DOUBLE | 预算金额 |
| 预算差异 | DOUBLE | 实际-预算差额（含缺失值） |
| 支付方式 | VARCHAR | 银行转账/支付宝/微信/信用卡/现金 |
| 审批人 | VARCHAR | 审批人员（含缺失值） |
| 是否固定支出 | BOOLEAN | 固定/非固定 |
| 季度 | VARCHAR | Q1-Q4 |
| 年份 | INTEGER | 2023-2025 |

**适合测试**: 部门费用对比、预算执行率、季度趋势、费用类别分析、审批流程分析

---

## 数据特征

- **缺失值**: 每个数据集 1-2% 随机缺失，模拟真实场景
- **异常值**: 数值字段注入 2-3% 极端值，用于异常检测测试
- **类别字段**: 地区、产品、行业等维度可用于分组聚合
- **日期字段**: 覆盖 2020-2025 年，支持时间序列分析

## 使用方式

```bash
# 生成数据
python testExcel/generate_test_data.py

# 启动应用后，通过文件上传界面上传 Excel 文件
streamlit run frontend/app.py
```
