## Integration

### How to Use

1. **Backend logs**: Structured JSON logs appear in uvicorn stdout
2. **Request tracing**: Use `X-Request-ID` header to correlate logs
3. **Performance monitoring**: Watch `X-Response-Time` header
4. **Slow query alerts**: Logs WARNING for queries > 2s

### Example Log Output

```json
{"time":"2026-05-23T20:00:00","level":"WARNING","logger":"enterprise_ai.observability","message":"Slow request POST /api/query → 200 (3500ms) [moderate]"}
```
