## 模块职责

### frontend-react/ — React 前端 (Next.js 15 + React 19)
- **panels/**: 工作区面板 (SQL, History, Preview, Upload, Tables, Status)
- **components/**: 通用组件 (Monaco Editor, DataTable, Export, Explain)
- **stores/**: Zustand 状态管理 (6 stores)
- **services/**: API 服务层
- **i18n/**: 国际化 (en/zh)

### backend/ — FastAPI 后端 API
- **routes/**: API 路由 (query, tables, upload, quality, table_manage)
- **services/**: 业务逻辑 (data_service, query_history)
- **models/**: Pydantic 数据模型

### database/ — DuckDB 数据库层
- **db_manager.py**: DuckDB 连接管理 (singleton)
- **query_executor.py**: SQL 查询执行 + EXPLAIN
- **file_loader.py**: CSV/Excel 文件加载
- **schema_detector.py**: 模式检测与类型映射
- **data_quality.py**: 数据质量分析引擎

## 技术选型

| 技术 | 用途 | 理由 |
|------|------|------|
| Next.js 15 | 前端框架 | App Router, SSR/SSG, API proxy |
| React 19 | UI 框架 | 企业级生态, 组件化 |
| Zustand | 状态管理 | 轻量, persist 中间件 |
| TanStack Table + Virtual | 虚拟化表格 | 大数据量性能 |
| Monaco Editor | SQL 编辑器 | 企业级代码编辑, 自动补全 |
| TailwindCSS | 样式 | 原子化, 暗色/亮色主题 |
| FastAPI | 后端框架 | 异步, 自动文档, 高性能 |
| DuckDB | 分析数据库 | 嵌入式, 高性能 OLAP |
